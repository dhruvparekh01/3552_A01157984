  6) Observer
  7) Strategy
  8) Bridge
  9) Decorator
  10) Facade
  11) State
  12) Mediator
  13) Proxy
  14) Lazy Initialization
  15) Dependency Injection
* Concurrency by:
  1) Multithreading
  2) Multiprocessing
  3) Asynchronous Programming
